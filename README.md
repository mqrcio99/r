# 📍 Consultor de CEP

Aplicação React para consulta de endereços através do CEP, utilizando a API ViaCEP.

## 🚀 Funcionalidades

- Busca de endereço por CEP
- Formatação automática do CEP (00000-000)
- Estados de loading e erro
- Interface responsiva e intuitiva
- Busca por Enter ou botão

## 🛠️ Tecnologias

- React
- CSS3
- API ViaCEP

## 📱 Como usar

1. Digite um CEP válido (8 dígitos)
2. Pressione Enter ou clique em "Buscar"
3. Visualize as informações do endereço

## 🔧 Como rodar

```bash
npm install
npm start
```

## 🎯 Aprendizados

- Consumo de APIs REST
- Gerenciamento de estados com React Hooks
- Validação de dados de entrada
- Tratamento de erros
- Design responsivo
